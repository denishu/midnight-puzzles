Privacy Policy

Last updated: May 11, 2026

Midnight Puzzles operates three Discord bots (Semantle, Travle, and Duotrigordle). This policy explains what data we collect, why, and how long we keep it.

What We Collect

  • Discord user ID and username — to track your game sessions and display your name in daily recaps.
  • Server (guild) ID — to post recaps and streaks to the correct server.
  • Channel ID — only when a server admin uses /setchannel to configure where daily messages go.
  • Game data — your guesses, scores, and completion status for each daily puzzle.

What We Don't Collect

  • Message content or chat history
  • Email addresses, IP addresses, or location data
  • User avatars or profile information beyond username
  • Data from users who don't interact with the bots
  • Data from other bots or apps in your server

How We Use Your Data

Your data is used exclusively to make the games work:

  • Tracking daily puzzle progress so you can resume across devices
  • Posting midnight recaps showing who played and their scores
  • Maintaining server streaks
  • Preventing the same puzzle from being played twice in one day

We do not sell, share, or transfer your data to any third parties. There are no analytics, ads, or tracking scripts.

Data Retention

  • Game sessions are automatically deleted after 7 days.
  • User records (Discord ID + username) are retained for streak and statistics continuity. These can be deleted on request.
  • Server configurations (channel ID) persist until the bot is removed from the server.

Data Storage

All data is stored in a private database on infrastructure we control. Data is not shared with external services. The bots communicate only with Discord's API.

Your Rights

You can request deletion of your data at any time by contacting us. Removing the bot from your server deletes all server-specific configuration. Game session data is automatically deleted after 7 days.

Contact

For questions or data deletion requests, reach out at playmidnightpuzzles@gmail.com
